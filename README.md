# 🚀 Proyecto Flutter + API Node.js + MySQL

Este repositorio contiene un proyecto completo compuesto por una **API en Node.js con MySQL** y una **aplicación desarrollada en Flutter**.

---

# 📂 Estructura del Proyecto

## 📁 Base de datos JS
Carpeta donde se encuentran las **APIs desarrolladas en Node.js** junto con el **script SQL de la base de datos**.

Incluye:
- Configuración del servidor
- Conexión a MySQL
- Endpoints de la API
- Script SQL para la base de datos

---

## 📁 login_registrorol_home
Proyecto **completo en Flutter**.

Incluye las siguientes funcionalidades principales:

- 🔐 Login de usuario  
- 📝 Registro de usuario  
- 👤 Creación de roles  
- 🏠 Vista Home principal  
- 🔗 Conexión con la API en Node.js  

---

# ⚙️ Tecnologías utilizadas

### Backend
- Node.js
- Express
- MySQL

### Frontend
- Flutter
- Dart

---

# ▶️ Cómo ejecutar el proyecto

## 1️⃣ Backend (API)

Entrar a la carpeta:

```bash
cd "Base de datos JS"
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el servidor:

```bash
npm run dev
```

o

```bash
npm start
```

---

## 2️⃣ Aplicación Flutter

Entrar a la carpeta:

```bash
cd login_registrorol_home
```

Instalar dependencias:

```bash
flutter pub get
```

Ejecutar el proyecto:

```bash
flutter run
```
